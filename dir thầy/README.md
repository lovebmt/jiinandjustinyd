# Thư mục tài liệu của thầy

Thư mục này dùng để lưu tài liệu tham khảo và hình mẫu do thầy gửi. Các hình trong đây chỉ dùng để tham khảo cách trình bày, không đưa nguyên vào bài Củ Chi vì khác đề tài.

- `huong_dan_dinh_dang/`: ảnh hướng dẫn cấu trúc và định dạng đồ án.
- `hinh_mau_cua_thay/`: các hình/biểu đồ mẫu về đề tài nguồn nhân lực du lịch.
- `tai_lieu_mau/`: file Word bài mẫu tham khảo nếu có.
