# Thư mục bài của tôi

Thư mục này chứa file gốc và các phần Markdown đã tách từ `nợ101m_2223.docx` để dễ quản lý, chỉnh sửa và cập nhật.

- `source/no_101m_2223.docx`: bản Word gốc được sao chép để lưu kèm.
- `markdown/00_INDEX.md`: chỉ mục toàn bộ các phần đã tách.
- `markdown/00_mo_dau/`: phần Mở đầu.
- `markdown/01_chuong_1/`: Chương 1.
- `markdown/02_chuong_2/`: Chương 2.
- `markdown/03_chuong_3/`: Chương 3.

Khi cần sửa bài, ưu tiên sửa từng file nhỏ trong `markdown/`, sau đó có thể ghép lại thành bản tổng hợp nếu cần.
